Documentation API

Documentation pour l'API d'analyse de sentiment d'un tweet.

Pour analyser le sentiment d'un texte, vous devez l'envoyer dans le paramètre 'text' d'un fichier json.

Exemple


# A curl example
curl -X POST https://tweet-sentiment-api-ociap7.herokuapp.com/api/sentiment_detection/ \
	-H 'Content-type: application/json' \
	-d '{"text" : "This example is awesome !"}'

Résultat


{
	result: "positive"
}

Erreurs

Vous pouvez obtenir une erreur 400 en cas de mauvaise requête :


{
	message: "Pas de document json dans le corps de la requête."
	documentation: "https://tweet-sentiment-api-ociap7.herokuapp.com/documentation"
}

Ce message indique que l'api n'a pas detecté de fichier json dans le corps de la requête. Plus d'informations ici : https://reqbin.com/req/c-dwjszac0/curl-post-json-example.


{
	message: "Le corps de la requête n'a pas de champs 'text'."
	documentation: "https://tweet-sentiment-api-ociap7.herokuapp.com/documentation"
}

Dans ce cas là, le fichier envoyé a été mal saisi, veillez faire attention à bien saisir la clé 'text'.
